# RINT Integrations Repository

Репозиторий интеграций для платформы RINT. Содержит заполненные YAML-файлы артефактов.

## Структура

```
{Integration.id}/                                         # UUID интеграции
└── v{semver}/                                            # Версия интеграции
    └── {GraphNode.name}/                                 # Имя узла
        └── {Template.name}-{Template.id}-v{semver}.yaml  # Артефакт
```

## Ветки (среды)

| Ветка     | Среда          | Описание                               |
|-----------|----------------|----------------------------------------|
| `dev`     | Development    | Разработка и начальное тестирование    |
| `test`    | Testing        | Функциональное тестирование            |
| `lt`      | Load Testing   | Нагрузочное тестирование (опционально) |
| `prelive` | Pre-production | Финальная проверка перед продом        |
| `prod`    | Production     | Продакшн                               |

## Git Flow

```
DEV → TEST → LT → PRELIVE → PROD  (с нагрузочным тестированием)
DEV → TEST → PRELIVE → PROD        (без нагрузочного тестирования)
```

Продвижение между средами осуществляется через Pull Requests.
